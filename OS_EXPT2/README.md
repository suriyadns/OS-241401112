# Experiment 2: Study of Unix editors: sed, vi, emacs

## Objective
To study and implement text processing using the Stream Editor (sed) and interactive text editors (vi and emacs).

## Summary of Tools
* **sed**: Used for automated search-and-replace and line deletion via script.
* **vi**: Demonstrated modal editing (Insert vs. Command mode).
* **emacs**: Demonstrated modeless editing and standard keybindings.

## How to Run the Script
1. chmod +x sed_demo.sh
2. ./sed_demo.sh
